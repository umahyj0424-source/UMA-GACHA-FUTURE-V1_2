# 우마무스메 가챠 미래시 플래너 v1.3

엑셀 `말딸 미래시 계산기 v6.1.1 (2026.05.21)의 사본.xlsx` 디자인을 웹으로 옮긴 정적 HTML/JS 사이트입니다.

## 구성

```text
index.html
assets/
  css/gacha-planner.css
  data/gacha-data.js        # v4 가챠 시뮬레이터 데이터: 캐릭터/서포트/배너
  data/planner-data.js      # 엑셀에서 추출한 초기 보유 상태, 가챠 예정 체크, 무료뽑 기본값
  js/gacha-planner.js       # 플래너 + 시뮬레이터 본체
```

## 주요 기능

- 사용자의 현재 상태 입력
  - 기준일, 현재 주얼, 말뽑/서폿 티켓, 한돌석 조각
  - 일일 예상 주얼, 월초 티켓 수, 구매 여부 메모
- 배너별 미래시 카드 표시
  - 시작/종료일, 픽업명, 캐릭터/서포트 구분
  - 시작 시 예상 주얼/티켓 계산
  - 무료뽑, 추가 수입 보정, 예정 주얼 사용 직접 입력
- `픽업 하지않음` 버튼
  - 해당 카드가 숨겨지고 다음 카드가 밀려 올라옵니다.
  - 상단의 `숨긴 가챠 보기`로 복구할 수 있습니다.
- 카드 옆 시뮬레이터
  - 1회/10회/100연차
  - 10연차 마지막 2성/SR 이상 보장
  - 무료뽑 → 티켓 → 주얼 순서로 소비 가능
  - 사용한 주얼/티켓/무료뽑이 즉시 플래너에 반영되어 다음 가챠의 보유량에서 차감됩니다.
  - `완료하고 다음` 버튼으로 다음 가챠로 진행합니다.
- 최고 등급만 집계
  - 캐릭터: 3★만 `지금까지 획득`에 표시
  - 서포트: SSR만 표시
- JSON 저장/불러오기
  - localStorage 자동 저장
  - 텍스트 JSON 내보내기/불러오기 지원

## 데이터 주의

업로드된 XLSX 내부의 `가챠 정보(평가x)` 시트는 `IMPORTRANGE` 캐시가 `#REF!` 상태입니다.
따라서 배너의 시작/종료일과 픽업명은 이전 v4 모듈에서 보정한 `gacha-data.js`를 우선 사용합니다.
무료뽑 수치는 숫자 캐시가 있으면 가져오고, 없으면 0으로 두었습니다.
단, 첫 오르페브르/삼틀딱 배너의 120 무료뽑은 첨부 스크린샷 기준으로 기본값 보정했습니다.

## 다른 사이트에 삽입

아래 세 파일을 기존 사이트에 추가하고, 원하는 위치에 `div`를 둔 뒤 mount 하면 됩니다.

```html
<link rel="stylesheet" href="assets/css/gacha-planner.css">
<script src="assets/data/gacha-data.js"></script>
<script src="assets/data/planner-data.js"></script>
<script src="assets/js/gacha-planner.js"></script>
<div id="gacha-planner-root"></div>
<script>UmaGachaPlanner.mount('#gacha-planner-root');</script>
```

GitHub Pages에서는 `index.html`과 `assets` 폴더 전체를 같이 올려야 합니다.


## v1.1 수정

초기 v1에서 `querySelector` 헬퍼 함수의 인자 순서가 섞여 브라우저에서 `root.querySelector is not a function` 오류가 발생할 수 있었습니다. 이 경우 헤더/JSON 박스만 보이고 미래시 카드와 입력칸이 렌더링되지 않았습니다. v1.1에서는 `$(root, selector)`와 `$(selector, root)`를 모두 허용하도록 수정했습니다.


## v1.2 수정

- 서클/팀레/챔미/말오스 입력칸을 텍스트 입력이 아니라 XLSX 데이터 유효성 목록 기반 `select` 드롭다운으로 변경했습니다.
- XLSX의 사료 시트들을 웹 데이터로 추출해 가챠 시작 시점 수입을 계산합니다.
  - 일일 미션, 데일리 쥬얼팩, 팀레/서클 보상, 시나리오/로그인, G1 미션, 신캐, 파카라이브, 챔미, 말오스, 먼슬리 매치, 트레이닝 패스, 이벤트, 레이싱 카니발, 레전드 레이스, 최강팀, 타키온, 기능시험, 기타 잡 사료, 마스터즈 챌린지, 월초 상점, 선택권/흑우팩 패키지를 포함합니다.
- 계산 범위는 엑셀 수식과 맞춰 `이전 기준일 초과 ~ 현재 배너 시작일 이하`로 합산합니다.
- 챔미/말오스/팀레/서클 보상은 드롭다운 선택값에 따라 PVP 보상표 기준으로 다시 계산합니다.
- 월초 상점은 월초 말뽑/서폿 티켓 입력값과 메달상점 구매 여부를 반영합니다.
